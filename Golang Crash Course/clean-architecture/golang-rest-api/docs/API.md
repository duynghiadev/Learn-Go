# API Documentation - Post Service

## Base URL

The base URL for all API endpoints is:

http://localhost:8081/api/v1
_(Replace `localhost:8081` if running on a different host or port)_

## Authentication

Currently, there is no authentication required to access the API endpoints.

## Endpoints

### Post Resource

#### 1. Get All Posts

Retrieves a list of all available posts.

- **Method:** `GET`
- **Path:** `/posts`
- **Request Body:** None
- **Success Response:**
  - **Code:** `200 OK`
  - **Content:** An array of Post objects.
  ```json
  [
    {
      "id": "firestore-generated-id-1",
      "title": "First Post Title",
      "text": "Content of the first post."
    },
    {
      "id": "firestore-generated-id-2",
      "title": "Second Post Title",
      "text": "Some interesting text here."
    }
  ]
  ```
- **Error Response:**
  - **Code:** `500 Internal Server Error`
  - **Content:**
  ```json
  {
    "error": "Failed to retrieve posts"
  }
  ```

#### 2. Create a New Post

Creates a new post entry in the database.

- **Method:** `POST`
- **Path:** `/posts`
- **Headers:**
  - `Content-Type: application/json`
- **Request Body:** A JSON object representing the post to be created. The `id` field should be omitted or left empty as it will be generated by the server.
  ```json
  {
    "title": "My New Awesome Post",
    "text": "This is the content of my new post. It's going to be great!"
  }
  ```
- **Success Response:**
  - **Code:** `201 Created`
  - **Content:** The newly created Post object, including the server-generated `id`.
  ```json
  {
    "id": "newly-generated-firestore-id",
    "title": "My New Awesome Post",
    "text": "This is the content of my new post. It's going to be great!"
  }
  ```
- **Error Responses:**
  - **Code:** `400 Bad Request` (e.g., validation failed)
  - **Content:**
  ```json
  {
    "error": "Post title and text cannot be empty"
  }
  ```
  - **Code:** `400 Bad Request` (e.g., invalid JSON)
  - **Content:**
  ```json
  {
    "error": "Invalid request payload"
  }
  ```
  - **Code:** `500 Internal Server Error`
  - **Content:**
  ```json
  {
    "error": "Failed to save the post"
  }
  ```
